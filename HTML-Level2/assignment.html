Learn about video tag

The main difference between Video and iframe tag is that the <video> tag is used to directly embed a specific video file into your page, while the <iframe> tag is used to embed an entire, separate webpage. The <video> tag is for showing a video you have the file for (like an .mp4), giving you native browser controls. An <iframe> is for showing content from another website, like a YouTube video player or a Google Map.
