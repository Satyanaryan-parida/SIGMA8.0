Convert each RGB value to hexadecimal

Given RGB:
rgb(32, 166, 166)

Step 1: Convert each RGB value to hexadecimal
1. Red: 32
Divide by 16: 32 ÷ 16 = 2 remainder 0

In hex: 2 → 2, 0 → 0

So, 32 → 20

2. Green: 166
Divide by 16: 166 ÷ 16 = 10 remainder 6

10 in hex = A

So, 166 → A6

3. Blue: 166
Same as above → A6

Step 2: Combine the hex values
Just concatenate the three hex pairs:
Red    Green   Blue  
 20  +  A6   +  A6  =  #20A6A6

Final Hex Code:
#20A6A6
